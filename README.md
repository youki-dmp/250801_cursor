# Cursor Git Project

このプロジェクトは、Cursor IDEとGitHubの連携をテストするためのプロジェクトです。

## 機能
- MCP（Model Context Protocol）によるGitHub連携
- 自動化されたGit操作
- AI支援によるコード開発

## セットアップ
1. GitHub CLIの認証
2. MCPサーバーの設定
3. 環境変数の設定

## 使用方法
```bash
# リポジトリのクローン
git clone https://github.com/youki-dmp/cursor_git.git

# 依存関係のインストール
npm install

# 開発サーバーの起動
npm start
```
